Define a module-level constant:
```
const FINANCE_USER_ID = "00000000-0000-0000-0000-000000000001";
```
This is the hardcoded UUID used for user_id on all Finance records (no real auth in this app). Per user decision, we use the admin client to bypass RLS.

GET handler:
- Call createAdminClient() from "@/lib/supabase/admin"
- Query supabase.from("invoices").select("*").order("created_at", { ascending: false })
- Return NextResponse.json(data || []) on success
- Return NextResponse.json({ error: error.message }, { status: 500 }) on error

POST handler:
- Call createAdminClient()
- Parse request body — expect: { client_name, client_vkn, description, gross_amount, kdv_rate, kdv_amount, stopaj_rate, stopaj_amount, net_amount, invoice_type, status }
- Insert into invoices table with user_id: FINANCE_USER_ID — do NOT include id (let Supabase auto-generate UUID)
- Use .select().single() after insert to get the created row back
- Return NextResponse.json(data, { status: 201 }) on success
- Return NextResponse.json({ error: error.message }, { status: 500 }) on error

Import: import { NextResponse } from "next/server"; import { createAdminClient } from "@/lib/supabase/admin";
